## アプリの概要
新規登録・ログインの機能があり、ログインしたユーザーのみが文章を投稿できるWebアプリ

## 使用技術
フロントエンド

・next.js

・TypeScript


バックエンド、DB、認証

・Firebase

認証にはFirebase Authentication, データベースにはFirestoreを使用


テストコード

・jest
```bash
npm test
```
